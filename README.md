
# Iran–Israel Conflict: Financial Market Impact (Case Study)

## Overview
This project analyzes the short-term impact of geopolitical tensions between Iran and Israel in early 2026 on global financial markets.

## Objective
To examine how key asset classes responded to geopolitical risk using real-time market data.

## Assets Analyzed
- S&P 500 (US Equity Market)
- NIFTY 50 (Indian Equity Market)
- Gold (Safe Haven Asset)
- Brent Crude Oil (Energy Commodity)

## Methodology
- Short-term trend-based analysis
- 10–15 day observation window (Late Feb – Mid March 2026)
- Data sourced from Yahoo Finance
- Focus on price movements and volatility patterns

## Key Insights
- Oil prices showed the strongest reaction to geopolitical uncertainty
- Equity markets displayed short-term volatility with varying regional sensitivity
- Gold maintained its role as a safe-haven asset
- Market behavior followed a phased response: initial caution → peak volatility → stabilization
